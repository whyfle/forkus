# 📦 Forkus Inventory

A simple inventory management web app built with Python and FastAPI.

## 🛠️ Tech Stack

- **FastAPI** — web framework
- **SQLAlchemy** — database ORM
- **Pydantic** — data validation
- **Uvicorn** — ASGI server
- **Python 3.x**

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/yourusername/inventario.git
cd forkus
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
uvicorn app.main:app --reload
```

Then open your browser at `http://localhost:8000`
